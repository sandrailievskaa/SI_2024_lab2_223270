Цикломатска комплексност

Цикломатска комплексност = (Број на региони) или (број на ребра - број на јазли + 2) или (број на предикатни јазли + 1)

број на ребра (E) = 40


број на јазли (V) = 32

 
Цикломатска комплексност = E - V + 2 = 40 - 32 + 2 = 10
